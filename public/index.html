API de Localizaciones de Colombia

Consulta los departamentos y ciudades de Colombia.

Departamentos

    Ciudades

      Cómo usar la API

      Obtener todos los departamentos

      fetch('/api/departments')
.then(response => response.json())
.then(departments => console.log(departments));

      Obtener todas las ciudades

      fetch('/api/cities')
.then(response => response.json())
.then(cities => console.log(cities));

      Filtrar ciudades por departamento

      fetch('/api/cities?departmentId=1')
.then(response => response.json())
.then(cities => console.log(cities));

      Desarrollado como API pública para consultar los departamentos y ciudades de Colombia.
